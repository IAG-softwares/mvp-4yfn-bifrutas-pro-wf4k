# BiFrutas Pro

Professional marketing platform for beverage businesses to showcase bifruit products, capture leads, and track marketing performance with comprehensive analytics and campaign management tools.

## Quick Start

```bash
# Install dependencies
npm run setup

# Start development servers
npm run dev
```

The frontend will be available at http://localhost:5173
The backend API will be available at http://localhost:3001

## Project Structure

```
├── frontend/          # React + Vite frontend
│   ├── src/
│   │   ├── pages/    # Page components
│   │   ├── components/ # Reusable components
│   │   └── api/      # API client
│   └── ...
├── backend/           # Express.js backend
│   ├── routes/       # API routes
│   ├── database.js   # SQLite database
│   └── server.js     # Express server

```

## Features

Dynamic product showcase builder, customer lead capture and management, marketing campaign tracking, product inventory management, sales conversion analytics, mobile-responsive landing page templates, social media integration, email marketing automation, and comprehensive reporting dashboard for marketing ROI analysis.

## Tech Stack

- **Frontend**: React 18, Vite, Tailwind CSS
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **Styling**: Tailwind CSS
